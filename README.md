# Chladni Patterns – Festival Interactive Installation 
This project aims to create an interactive installation demonstrating Chladni patterns for a public festival.

Chladni patterns appear when sound vibrations move through a rigid surface covered with fine particles (such as sand). At certain frequencies, the particles arrange themselves into beautiful geometric shapes, making sound waves visible to the human eye.

Our goal is to build a reliable, visually appealing, and educational system that allows visitors to:

- Observe sound waves in real time
- Experiment with different frequencies
- Discover the relationship between physics, music, and geometry
- Interact with a scientific phenomenon in a creative way

## Objectives

- Design and build a functional Chladni plate system
- Provide a safe and robust installation for public use
- Create an engaging educational experience

## How Chladni Patterns Work

A flat plate is attached to a vibration source (speaker or actuator). When the plate vibrates at specific frequencies:

1. The plate oscillates in complex modes
2. Some areas remain still (nodes)
3. Particles accumulate on these node lines
4. Geometric patterns appear

Chladni patterns appear at specific resonance frequencies, which depend on the geometry, size, material, and boundary conditions of the plate. These frequencies correspond to vibration modes whose wavelengths "fit" the plate dimensions.
